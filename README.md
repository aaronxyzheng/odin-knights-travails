# Knight Moves

Finds the shortest path for a knight to move between two squares on a chess board using a Breadth-First-Search (BFS) technique.

## Usage
```javascript
shortestPath([0, 0], [7, 7]);
```

**Output:**
```
You can make it in 6 moves!
[0, 0]
[2, 1]
[4, 2]
[6, 3]
[4, 4]
[6, 5]
[7, 7]
```
